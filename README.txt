Requirement
 
The web page will display 3 lights (Red, Yellow, Green) on the center of the page, 

Add 2 buttons to start and stop the simulation.
 
Initially all the lights are in OFF position.
 

First light is RED, second is YELLOW, third is GREEN.

User clicks on the "Start" button, which starts the simulation.


RED light will turn ON.

Then every 10 seconds the next light will turn ON and current ON light will turn OFF.

This stops when user clicks on the Stop button.

 

Light Positions:

ON – Display light color

OFF – Display circle without any color
